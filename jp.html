Single Page Ecommerce Website

Built with:

  • Angular
  • Bootstrap
  • ASP.NET Core
  • Microsoft Azure SQL
  • Microsoft Azure DevOps

NotAmazon.com is a fictional online tech store where customers can browse and purchase items from a variety of sellers. The site consists of a customer-facing view, a seller-facing view, and an administrator-facing view, all three of which being fully functional. The site provides all of the features expected from a modern ecommerce website, including search, messaging between customers and sellers, and item recommendations. Transactions are handled securely through the PayPal API, and the backend makes use of the Argon2 algorithm for password hashing.

Graph Based Dialogue Editor

Built with:

  • Godot
  • C# (Mono)

This project is a graph based dialogue editor created to streamline the writing of dialogue for interactive fiction and other applications. Scenes can be exported to JSON, and individual story paths can be exported to a format similar to that of a film script. Custom attributes can be created, such as for background images or audio cues, to allow for more seamless integration with the target application upon export. The application also features a robust spellchecker, a feature that has demonstrated itself to be extremely convenient when writing more long-form content.

Networked Physics Tech Demo

Built with:

  • Unity
  • C#

This project is a real-time networked physics demo where multiple clients control and interact with physics objects in a shared physics environment. Various interpolation and client authority swapping techniques were used to make physics interactions seem smooth and instantaneous across all clients, even with latency of up to 1000ms. This video shows clients approximately 350ms away from the host still experiencing smooth, consistent, and responsive movement, far higher than what is considered optimal for most applications making use of real-time networked physics.
